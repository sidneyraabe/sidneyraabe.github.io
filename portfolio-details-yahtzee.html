"Five Dice" Console Application

  1. Home
  2. Project Details

Project information

  • Category: Console application
  • Project date: September 2020
  • Project URL: DiceGame

After an assignment to make a rock paper scissors game in my programming course, I had the inclination to make a fully functional "Five Dice" game. Considering how much programming I knew at the time, this was a huge stretch. However, everything works, and looks snazzy as a bonus. :)

Features:

  • Attractive UI
  • Layered architecture, separate BLL and UI layers
  • Thorough user validation
  • In-memory data storage

    • I definitely need to revisit and refactor this program now with one year of coding knowledge under my belt, but I'm really proud of how pretty everything looks.

    Improvements:

  • Refactor/unit test BLL methods

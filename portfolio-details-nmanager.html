  1. Home

Project information

  • Category: Console Application
  • Project date: October, 2020
  • Project URL: NounManager

This is a simple console CRUD application which keeps track of a collection of books as well as three properties attributed with each book.

Features:

  • In-memory data storage
  • Layered Architecture
    • Controllers layer
    • Models layer
    • Data layer
    • View layer
  • User validation

Despite being an early project, I don't have to many complaints about it.

Improvements:

  • UI could use some tweaking in order to display long strings without breaking columns
  • Add/edit screens could use occasional Console.Clear(); methods to refresh screen
  • Unit testing implementation
  • General UI pretty-ifying
